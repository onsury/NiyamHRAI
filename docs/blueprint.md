# **App Name**: NiyamAI

## Core Features:

- Founder Philosophy Intake: A conversational interface, powered by Claude API, gathers the founder's philosophy, probing decision-making, risk appetite, and cultural expectations to establish the CorePersonaDNA.
- AI-Powered Persona DNA Mapping: Maps employee traits to the 67-trait behavioral framework using Gemini API. Converts qualitative skills data into quantitative neural signature to enable organisation to achieve founder-employee behavioural allignment
- Real-time Behavioral Drift Detection: Continuously monitors and analyzes employee check-ins, via Cloud Functions, to identify and alert users to any deviations from the founder's CorePersonaDNA, displaying this data with Recharts line charts.
- Personalized Mentorship via AI: Generates tailored mentorship insights and recommendations using Claude API to help employees course correct and improve neural alignment. Leverages data accumulated in weekly check-ins.
- Neural Honing Lab Simulations: The LLM Gemini API is used as a tool to generate personalized, role-play simulations for employees, providing them opportunities to apply their understanding of the founder's principles.  Then LLM Claude API assesses user responses and scores their synergyDelta and provides tailored feedback for optimal skill growth
- HR Neural Insights Dashboard: Compiles org-wide engagement data, identifying at-risk teams or departments, then uses Gemini and Claude to recommend proactive interventions, presenting key insights visually.
- Firestore Integration: Stores and retrieves organizational data, user profiles, competency frameworks, and AI-generated insights to create an Employee Growth Platform.

## Style Guidelines:

- Primary color: Navy (#1B2541) for headers and primary text, reflecting a sense of stability and leadership.
- Accent color: Indigo (#6366F1) for interactive elements, links, and highlights, providing a modern, technological touch.
- CTA color: Amber (#F59E0B) for call-to-action buttons and important metrics, drawing attention to crucial actions and information.
- Background color: White (#FFFFFF) and Slate (#F8FAFC) to create a clean and professional canvas, with dark navy sections for contrast.
- Headings: Bold uppercase 'Space Grotesk' sans-serif with tracking-tight to provide a clear, modern, technological feel.
- Body: Clean 'Inter' sans-serif font, ensuring legibility and a contemporary look.
- Use a consistent set of icons for navigation and feature representation. The target emoji denotes 'Alignment Over Compliance,' diamond represents 'First Principles Design,' and lightning depicts 'Growth Velocity'.
- Dashboard with a left sidebar navigation (navy background), a main content area, and a right sidebar for metrics. Cards use large rounded corners (rounded-3xl to rounded-[48px]), subtle borders, and generous padding (p-8 to p-12).
- Fade-in with slide-up on page load, scale on hover for interactive elements, and pulse animations for live data updates, enhancing user engagement and feedback.
